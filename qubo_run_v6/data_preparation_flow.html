4Per-donor pseudobulk aggregation (per cell type)
各 (donor, cell type) について: 同 cell type の全細胞にわたり遺伝子発現を 平均化 (mean aggregation) → donor × gene 行列 を cell type ごとに構築
統計単位: donor (人) cell type 情報を保持 ≥ 20 cells/donor 必須 感度解析: sum aggregation
→ pseudoreplication を回避 (Squair et al. 2021)、cell type 別の signal を温存
▼
5Highly Variable Gene (HVG) selection
全 32,170 遺伝子から発現分散上位 ~3,000 HVG を抽出 (Seurat FindVariableFeatures, vst method)
▼
6Biology filter (evidence-based, pre-DEG)
除外: (i) ミトコンドリア (MT-, MTRNR…), (ii) リボソーム (RPL[0-9], RPS[0-9], MRPL, MRPS), (iii) 熱ショック (HSPA/B/D, HSP90), (iv) 核内 lncRNA (MALAT1, NEAT1), (v) housekeeping (ACTB, GAPDH, B2M, FAU, EEF1A1), (vi) X-inactivation (XIST, TSIX), (vii) 低信頼 loci (AC*, AL*, LINC, MIR*, RNU*, SNORA, SNORD)
意図的保持: RPLP0/1/2 (specialized ribosome / Genuth & Barna 2018), RPSA (= laminin receptor, BBB transmigration / Nelson 2008), Tier 3 翻訳因子 (TPT1, EEF2, EIF*, TMSB*, HNRNPH1)
準拠: Heumos 2023, Luecken 2019
▼
7Differential expression analysis — edgeR (primary)
Per cell type × per holdout cohort で MS vs HD の DEG 解析。 モデル: ~Dx + log10(n_cells) + age + sex + batch
感度解析: DESeq2, limma-voom も並列実行 — 主要シグナルが手法に依存しないことを確認
▼
8Top-N candidate gene pool (per cell type)
各 cell type で |t-statistic| 上位 100 遺伝子を候補プールとして QUBO に渡す
Biology filter 通過後 → 約 835 個の cell-type-specific 候補遺伝子が QUBO 入力 (8 cell type × ~100, 重複除く)
▼
9Per-cell-type QUBO selection
H(x) = −s'x + γ x'R x + λ (Σx − K)² — relevance × non-redundancy × cardinality を同時最適化、Simulated Annealing で求解。
K ∈ {10, 20, 30} 内側 5-fold CV で K 自動選択 SA reads=30, sweeps=600 ~3 sec/instance
結果: 平均 K ≈ 17 遺伝子/cell type、3 cohort × 5 fold で 448 unique genes (union)、49 stable genes (≥50% panels)
▼
10Cell-type classifier + Soft-voting ensemble
各 cell type で L2 logistic regression を選択遺伝子で訓練 → MS 確率を出力 → 8 cell type の確率を 平均 (soft voting) し donor 単位の最終予測
