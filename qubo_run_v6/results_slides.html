研究事例①
遺伝子選択手法の比較 — 5 種類のモデル原理
  • scRNA-seq の donor 単位 pseudobulk から MS / HD 判別を行うため、5 種類の遺伝子選択手法を比較
  • 4 つは標準的手法 (DE_top, HVG, LASSO, Elastic Net)、1 つは提案手法 (QUBO)
手法 原理 選択メカニズム 特徴
DE_top edgeR の |t| 統計量 上位 K 個を選択 単純ソート 最もシンプル
HVG train donor 間の 発現分散 大の遺伝子 K 個 教師なし label 不使用
LASSO L1 正則化付きロジ回帰
min L(w) + λ Σ|w_j|		 係数を 0 に縮める スパース
Elastic Net L1 + L2 のハイブリッド正則化 LASSO の安定版 共線性に強い
QUBO 提案手法 二次最適化 (relevance + redundancy + cardinality 同時最適化):
min ( −s'x + γ x'Rx + λ(Σx − K)² )		 Simulated Annealing 冗長性を明示的にペナルティ

QUBO のみが 「relevance(関連度) と redundancy(冗長性) を同時最適化」 する点で他 4 手法と本質的に異なる

CSF (髄液) における判別性能 — 3 cohort 平均
  • 3 つの cohort を順番に hold-out として外部テストに用いる Leave-One-Cohort-Out (LOCO) 評価
  • 5-fold donor-level GroupKFold で内部 CV、(K, λ, γ) は inner CV AUC で fold ごとに最適化
  • edgeR DEG + 8 cell type per-cell-type ensemble (soft voting)

① AUC mean: QUBO 0.781 (vs LASSO 0.773)

② F1 mean: QUBO 0.640 (vs LASSO 0.620)

③ MCC mean: QUBO 0.297 (vs LASSO 0.242, +23%)

④ AUC σ: QUBO 0.041 (vs LASSO 0.057)

AP のみ LASSO に僅差負け (Δ 0.013)。主指標 AUC で勝、補助指標 MCC で大差勝。

研究事例④
パイプライン進化 — v5 → v6 で QUBO が LASSO に追いついた
  • 初期 v5 では QUBO が LASSO に劣っていたが、3 つの改良で同等性能まで到達
  • ① per-cell-type panel + 8-way ensemble、② cell type ごとの K グリッドサーチ、③ anti-batch 項
パイプライン アーキテクチャ 遺伝子数 Held AUC Held σ Held MCC
v5 Tier 1 全細胞種統合 1 panel + 1 分類器、housekeeping 残し 20 (固定) 0.781 0.062 —
v5 Tier 2B 同上 + anti-batch 項 (cohort 分散ペナルティ) 20 (固定) 0.728 0.143 —
v6 (no bio) Per-cell-type panel + 8-way ensemble + K グリッド (housekeeping 含む) ~87 (unique) 0.850 0.088 —
v6 + biology filter final 同上 + housekeeping/mito/ribosomal を早期除外 ~75 (unique) 0.781 0.041 0.297

最終 (v6 + biology filter) で AUC 0.781 ± 0.041, MCC 0.297 を達成。AUC mean / σ / F1 / MCC で QUBO が首位 (4/5 metrics 勝利)

4 つのアーキテクチャ改良

① Per-cell-type panel: 8 細胞種で独立に QUBO → cell-type-specific marker
② K grid search: K ∈ {10, 20, 30} を inner CV で選択 → cell type ごとに最適 panel size
③ Soft voting ensemble: 8 分類器の予測確率を平均 → cell type 欠損に頑健
④ Biology filter (早期適用): housekeeping / mito / ribosomal を選択前に除外 (post-hoc bias 回避)
