Background
なぜ MS の cross-cohort 分類が課題か
  • 多発性硬化症 (MS) は世界で約 280 万人 が罹患する難治性自己免疫疾患
  • scRNA-seq による 診断バイオマーカー研究が活発化している一方、 施設間で実験条件が異なると判別精度が崩壊する問題が顕在化
10x chemistry version
v2 / v3 / v3.1 で UMI 分布が変動
Cryopreservation
凍結 vs fresh で生細胞構成が変化
Sampling protocol
CSF 採取量、密度勾配分離の差異
既存手法の限界
  • DE-top (top-|t| ranking): 同一経路の冗長な遺伝子を選びがち、cohort 依存性が高い
  • LASSO / Elastic Net: 遺伝子数 K の制御が 正則化パラメタ λ の間接調整 しかなく、ぴったり K=20 が出せない
  • HVG (高変動遺伝子): 教師なしのため batch effect で動く遺伝子も拾ってしまう
  • Cell-level Wilcoxon (Seurat::FindMarkers): pseudoreplication 問題で false positive 過大 (Squair et al. 2021)
本研究のアプローチ
  • Cell type 別 QUBO 最適化: 8 つの免疫細胞種で独立にパネル設計
  • Relevance + Redundancy + Cardinality を同時最適化: 既存手法では同時に扱えなかった 3 軸
  • 3 cohort × LOOCV で外部汎化を直接評価: 別施設でも安定するパネルか定量化
  • Biology filter 早期適用: housekeeping/mito/ribosomal を選択前に除外、選択遺伝子の生物学的妥当性を担保
▶ 提案: QUBO による cell-type 特異的遺伝子パネル設計フレームワーク
「relevance × non-redundancy × cardinality」を同時最適化することで、 cross-cohort で再現可能な MS バイオマーカーパネルを構築する。
