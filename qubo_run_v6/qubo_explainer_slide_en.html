How QUBO Selects Genes
Choosing the "right K=20" via combinatorial optimization

Beyond simple ranking (DE_top) or LASSO: QUBO simultaneously optimizes "informative × non-redundant × exactly 20"

H(x) = −s⊤x + γ · x⊤R x + λ (1⊤x − K)2
① Maximize relevance
Each gene's "informativeness" si (e.g., |t-stat|²) is summed over selected genes — favoring genes that differ between MS and HD donors
② Minimize redundancy
Pairwise correlations Rij = |corr| are penalized for selected pairs — discouraging the selection of mutually similar genes (ensures diversity)
③ Exactly K genes
(Σxi − K)² drives the count toward K. Both over- and under-selection incur a penalty

x ∈ {0,1}n: binary variables (1 = select gene, 0 = drop). s and R are estimated from training donors only. The minimizer x* of H(x) yields the optimal K-gene panel.

Solver: Simulated Annealing
x = random initial state (K ones)
T = T_start (high; broad exploration)
while T > T_end:
   for i in all genes:
      ΔH = change in H if i is flipped
      if exp(−ΔH/T) > rand():
         flip x[i] # accept
   T *= 0.99 # cool down
# High T: occasionally accept worsening moves
# (escape local optima)
# Low T : only accept improving moves
# (converge to optimum)

At high temperatures the solver climbs over barriers (escape local optima); slow cooling drives convergence to a global solution.

Why QUBO?
  • Explicit redundancy control: Top-|t| ranking (DE_top) often selects multiple genes from the same pathway. QUBO ensures pathway diversity
  • Direct cardinality constraint: "Exactly K = 20" is built into the objective function (LASSO controls K only indirectly via λ tuning)
  • Tunable trade-off: γ and λ allow the relevance / redundancy / cardinality balance to be tuned per cell type
  • Quantum-annealer compatible: As a binary optimization, the same objective can be deployed on quantum annealers (e.g., D-Wave) without modification. Classical SA already runs in ~3 s per instance
  • Empirical advantage: In our study, QUBO achieved AUC 0.781 ± 0.041 — best on AUC, F1, MCC, and lowest cohort variance
