なぜ MS の cross-cohort 分類が課題か
  • 多発性硬化症 (MS) は世界で約 280 万人。scRNA-seq による診断バイオマーカー研究が活発化
  • しかし 施設間で実験条件 (10x version, 凍結, sampling) が異なると判別精度が崩壊する問題が顕在化
  • 既存手法 (DE-top / LASSO / HVG) は 冗長性制御や cardinality 制約を直接扱えず 、cross-cohort 再現性に限界

本研究: QUBO による cell-type 特異的遺伝子パネル設計 + 8-way ensemble + biology filter で cross-site 再現性を実現

Method
QUBO による遺伝子選択
H(x) = −s⊤x + γ · x⊤R x + λ (1⊤x − K)2
① relevance を最大化
各遺伝子の効きそう度 si (= |t-stat|²) を選んだ遺伝子で総和 — MS と HD で差がある遺伝子を優先
② redundancy を最小化
遺伝子ペアの相関 Rij を選んだペアで罰則 — 似た遺伝子の重複を回避 (多様性確保)
③ ちょうど K 個
(Σxi − K)² でカーディナリティ制約。解法: Simulated Annealing (~3 秒/instance)
5 手法を K=20 で比較

6
Conclusion
Take-home Messages
1
性能 — 主要 3 指標で 1 位
CSF において AUC 0.788、 F1 0.635、MCC 0.258。 AUC・F1・MCC の 3 指標で 1 位 (AP・σ_AUC は Elastic Net が僅差で 1 位)。
2
Cross-site 再現性 — Elastic Net と互角
3 cohort 間 std が 0.044 (Elastic Net 0.041, HVG 0.048, LASSO 0.068)。 Pappalardo / Heming / Ramesh で 0.74-0.82 のタイト範囲に収束し、外部 cohort で安定。
3
生物学的妥当性 — MS 病態と整合
高頻度選択遺伝子: HLA-DRB1, HLA-DPB1 (MS GWAS), ISG15 (Type I IFN), GZMA (細胞傷害), FTH1/FTL (鉄代謝)。 GO 解析: MHC II 抗原提示 p < 10⁻⁸。
4
量子計算機 互換
二値最適化として定式化されているため、D-Wave 量子アニーラにそのまま実装可。 現状は古典 SA で ~3 秒/instance、量子ハードウェア進歩時にシームレス移行。
Implication for Clinical Translation
QUBO は 「最高 AUC × 最高 F1・MCC × Elastic Net と互角の cross-cohort 安定性 × 生物学的妥当性 × 量子互換」 を同時に満たし、cross-site 再現性ある MS biomarker パネル として臨床応用に向けた強い候補となる。
▶ Future work (Phase 2): Multi-Instance Learning (MIL) による cell-level 拡張。各 donor を bag、cells を instances として attention 機構で集約し、「どの細胞 subset が病態に寄与したか」を解釈可能に。 QUBO は (i) gene selection の役割を維持しつつ、(ii) 各 donor から informative cells の coreset 選択という新たな役割 も担い得る。 Joint gene-cell QUBO (~5,000 binary vars) は D-Wave Leap Hybrid CQM Solver で実装予定 — 量子計算機実証としても位置付ける。
