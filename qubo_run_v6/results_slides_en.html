Method Principle Summary
Comparison of Five Gene-Selection Methods
  • For donor-level MS / HD classification from scRNA-seq pseudobulk, we compared five gene-selection methods
  • Four are established baselines (DE_top, HVG, LASSO, Elastic Net); one is the proposed method (QUBO)
Method Principle Selection Mechanism Characteristic
DE_top Top-K genes ranked by |t| statistic from edgeR Simple ranking Simplest baseline
HVG Top-K genes by expression variance across train donors Unsupervised Label-agnostic
LASSO L1-regularized logistic regression
min L(w) + λ Σ|wj|		 Coefficient shrinkage Sparse
Elastic Net L1 + L2 hybrid regularization Stabilized LASSO Robust to collinearity
QUBO Proposed Quadratic binary optimization (simultaneously balances relevance + redundancy + cardinality)
min ( −s'x + γ x'Rx + λ(Σx − K)² )		 Simulated Annealing Explicit redundancy penalty

QUBO is the only method that simultaneously optimizes relevance and non-redundancy, distinguishing it from the four standard methods

Result 1 / Main Performance
Classification Performance on CSF — Mean across 3 LOOCV Folds
  • Leave-One-Cohort-Out (LOCO) evaluation: each of three cohorts is sequentially used as the external held-out test set
  • Within each LOCO fold, 5-fold donor-level GroupKFold; (K, λ, γ) tuned by inner-CV AUC per fold
  • edgeR DEG layer + per-cell-type ensemble across 8 immune cell types (soft voting)

QUBO Wins on 4 of 5 Metrics

① AUC mean: QUBO 0.781 (vs LASSO 0.773)

② F1 mean: QUBO 0.640 (vs LASSO 0.620)

Loses only AP to LASSO (Δ 0.013). Wins on the primary metric AUC, with a stronger lead on the imbalance-robust MCC.

Result 3 / Pipeline Evolution
Pipeline Evolution — How QUBO Caught Up to LASSO (v5 → v6)
  • In early versions QUBO underperformed LASSO; three architectural improvements closed the gap
  • ① Per-cell-type panels with 8-way ensemble   ② Cell-type-wise K grid search   ③ Anti-batch term
Pipeline Architecture Genes Held AUC Held σ Held MCC

Final pipeline (v6 + biology filter) achieves AUC 0.781 ± 0.041, MCC 0.297, with QUBO leading AUC mean / σ / F1 / MCC (4 of 5 metrics)

Four Architectural Improvements

① Per-cell-type panel: independent QUBO per cell type → cell-type-specific markers
② K grid search: K ∈ {10, 20, 30} chosen by inner CV → optimal panel size per cell type
③ Soft voting ensemble: probabilities from 8 cell-type classifiers averaged → robust to missing cell types
④ Biology filter (early): housekeeping / mitochondrial / ribosomal genes excluded BEFORE selection (avoids post-hoc bias)
