Method
Quadratic Unconstrained Binary Optimization (QUBO) による細胞種別遺伝子選択
▶ QUBO とは: バイナリ最適化のフレームワーク — 各遺伝子を選ぶ (1)/選ばない (0) として、目的関数 H(x) を最小化する。本研究の H(x) は以下の 3 項からなる:
H(x) = −s⊤x + γ · x⊤R x + λ (Σx − K)2
① 情報量の多い遺伝子を選ぶ
si = edgeR |t|²
② 相関の高い遺伝子に罰則
Rij penalty → 冗長性制御
③ パネルサイズ K を固定
K ∈ {10, 20, 30} 内側 CV 自動選択

Results 2 — Selected genes per cell type (CSF)
CSF における QUBO 選択遺伝子 — 選択安定性と細胞種別 panel ・ 検証パネル一覧
▶ 解釈: 各細胞種の panel が biologically coherent な program を反映。B = plasma / IgM (IGHM)、Mono = iron metabolism (FTL) + MHC II (HLA-DPB1)、NK = cytotoxic (KLRB1/KLRC1/CCL5)、dnT = cytotoxic + Type I IFN (GZMA, ISG15)、gdT = cytotoxic (GZMA, CD69)。CD4_T/CD8_T/DC は 安定した選択が得られず — heterogeneous な pseudobulk 集団における signal dilution を反映している可能性 (灰色列)。
  • 赤枠= MS 病態と関連する biology category (右側緑ラベル) に該当する遺伝子。頻度上位ではなく biology 基準で selection (例: Mono CST3 = 100% は一般的 cell identity marker のため除外)。
▶ 独立論文との照合 (外部検証): Ramesh ら 2020 年 PNAS の 病的 B 細胞シグネチャと比較したところ、候補プールに残った 13 遺伝子全てが QUBO によって独立に再現。CSF immune dynamics シグネチャも 9/12 を捕捉 → 選択された panel の biological validity を裏付け。
6
Conclusion
QUBO は biology と biomarker を同時に提供
1
Biomarker performance
QUBO_hybrid: CSF AUC 0.858、F1 0.731、MCC 0.427。F1 / MCC で 7 手法中 1 位 (imbalance-robust)、AUC でも DE_top に僅差で 2 位。
2
Biological validity
Mono は MHC II + iron metabolism、NK/dnT/gdT は cytotoxic axis、B 細胞は 免疫グロブリン関連 program を捕捉 — known MS-associated immune programs を再現。
3
Cross-cohort 再現性
3 cohort で σ_AUC 0.108 (supervised 手法中 top-2)。Heming (balanced) で 0.738、Pappalardo/Ramesh で 0.89-0.95 のタイト範囲。translational potential を支える性質。
4
Methodological novelty
私たちの知る限り、relevance × 非冗長性 × cardinality を同時最適化する細胞種別 gene panel 設計の最初期の枠組みの一つ。冗長性を明示的にモデリング — DE-top や標準的 sparse selection 手法とは異なる。
Implication
QUBO は MS scRNA-seq データに対して biomarker reproducibility と biological coherence を同時に評価する unified framework を提供する — translational potential と機構的解明の両方に資する。
▶ Future Work (Phase 2) — Resolving T-cell signal attenuation
  • Limitation: Pseudobulk aggregation attenuates key T-cell signals (CD4 Th17, CD8 cytotoxic)
  • Approach: Multi-Instance Learning (MIL) — Each donor = bag of cells → Attention identifies disease-associated cell subsets (e.g., Th17-like, Trm/exhausted CD8)
  • Extension of QUBO: Gene selection → Cell coreset selection → Select informative cells per donor
