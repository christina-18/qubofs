# data_release/

Canonical result tables for the manuscript, generated by the pipeline
(`scripts/01_pipeline` → `scripts/04_aggregation`) under the run tag
`primary_bio_edger_counts` with the **matched fixed panel size K = 10** for all
methods. These files let readers check the main quantitative tables and figures
without re-running the full pipeline. All quantitative benchmark values are
deterministic under fixed seeds and match the manuscript and Supplementary
Information.

| File | Contents |
|---|---|
| `metrics_cross_cohort.csv` | Main Table 2: per-method ROC-AUC, σ_AUC, MCC, Macro-F1, Balanced Accuracy, within-panel \|ρ\| (means over the 3 held-out cohorts). |
| `metrics_per_cohort.csv` | Per-(held-out cohort × method) means and fold spread (Figure 2). |
| `selected_genes.csv` | quboFS per-cell-type selection frequency across the 14–15 cohort×fold panels (`cell_type, gene, selection_freq, n_panels`). Source for Figure 4 and Supplementary Table S6. |
| `within_panel_redundancy_summary.csv` | Mean within-panel \|ρ\| per method (n_panels). |
| `within_panel_redundancy_perpanel.csv` | Per-panel \|ρ\| (method × holdout × fold × cell type) — input to the paired permutation test. |
| `threshold_sensitivity.csv` | Fixed-0.5 vs inner-CV-tuned threshold metrics per method (Supplementary Methods section 4). |
| `permutation_tests.csv` | Paired sign-flip permutation tests, quboFS vs each baseline (Supplementary Table S3). |
| `bootstrap_ci.csv` | Bootstrap 95% CIs per method × metric (Supplementary Table S3). |
| `panel_size_sweep.csv` | Panel-size (K) sweep: per (K, method) ROC-AUC, MCC, Macro-F1, within-panel \|ρ\| (Supplementary Figure S2). |
| `solver_sensitivity.csv` | Simulated-annealing vs exact (brute-force) optimum per screened panel (Supplementary Figure S3, Supplementary Table S4). |
| `pbmc_pooled_redundancy.csv`, `pbmc_pooled_table1.csv` | PBMC generalisation benchmark (Supplementary Table S7). |

**Headline (matched K = 10):** quboFS has the **lowest within-panel redundancy of
any method** (|ρ| = 0.247; significantly below every baseline, paired permutation
all *p* < 0.001) while retaining competitive ROC-AUC (0.815, numerically highest
but not significantly different among the disease-informed methods; vs Elastic Net
*p* = 0.502). quboFS is a low-redundancy panel-design method, not an
accuracy-maximiser. These values match the manuscript Table 2 and are reproduced
by `metrics_cross_cohort.csv`.
