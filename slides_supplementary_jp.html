S6PBMC AUCell — cell-type × MS gene set 活性
PBMC の cell-type × MS gene set 活性 — AUCell

AUCell (Aibar 2017) で 7 種 MS gene set × 8 細胞種 (PBMC) の per-cell scoring。左: MS 細胞での活性、右: MS − HD 差分。CSF と比較すると PBMC では効果サイズ全般に小さく、CSF が MS 病態シグナルをより直接的に捕捉することの裏付け。

▶ 臨床的解釈

S7QUBO 定式化 — 詳細
QUBO (Quadratic Unconstrained Binary Optimization)
▶ QUBO とは: 「Yes/No (= 0/1) の選択問題」を 2 次関数で表現する最適化フレームワーク。 本研究では「各候補遺伝子を選ぶか (1) 選ばないか (0)」という選択問題を、 relevance (効く遺伝子) × 非冗長 (重複しない) × cardinality (個数) の 3 観点を含む単一のスコア関数で表現し、それを最小化する遺伝子組合せを探索します。 化学情報学・組合せ最適化分野で標準的な定式化で、Simulated Annealing 等の汎用 solver で解けます。
H(x) = −s⊤x + γ · x⊤R x + λ (Σixi − K)2
x ∈ {0,1}p: 候補プールの各遺伝子を選択 (1) か非選択 (0) で表すバイナリ変数 (p ≈ 100 / cell type)
① Relevance — −s⊤x
si = edgeR の |t-statistic|2 (MS vs HD)
→ MS と HD を区別する力が強い遺伝子ほど大きな score
Z-score 標準化で全遺伝子 N(0,1) に。
選ぶと H が下がる = 報酬。
② Redundancy — γ · x⊤R x
Rij = |Spearman corr(gi, gj)| (training pseudobulk 上)
→ 似た発現挙動の遺伝子ペアを同時選択すると罰則
γ は scale balancer。
多様性を強制 = LASSO/DE_top には無い。
③ Cardinality — λ(Σx − K)2
K ∈ {10, 20, 30} を 内側 5-fold CV で自動選択
→ K から外れると二次罰則 (soft constraint)
λ は relevance/redundancy より十分大きく。
panel size を後付けでチューニング不要。
▶ なぜ QUBO？
バイナリ変数 + 二次形式で書ける問題は GPU SA・古典 ILP・(将来的に) 量子ハードウェア の何れでも解ける汎用形式。本研究では候補数が 100 程度なので Simulated Annealing で約 3 sec/instance。将来 candidate を全 ~8,000 遺伝子に拡張する際は GPU 並列 SA で対応可能。
▶ K = 10 固定にしない理由
Cell type ごとに 最適 K が異なる (B cell は IGH/IGL ファミリで多様性が高く K=20-30、Mono は MHC II が支配的で K=10 で十分など)。K も 同じ inner CV でデータ駆動的に決めることで、cell type 別の biology に沿った panel size を実現。実測平均は ~17 遺伝子/cell type。
▶ 実装サマリ: qubo_pipeline_v6.py L89: K_GRID = [10, 20, 30] ／ Solver: dwave-neal SimulatedAnnealingSampler (10 reads/seed) ／ γ・λ: 内側 CV で grid 探索 (γ ∈ {0.5, 1.0, 2.0}, λ = 10·max(s)) ／ Final K: validation AUC で選択。

▶ なぜ最大細胞数の CD4/CD8 で DEG が立たないのか
① 機能サブタイプの混合体: CD4 は Th1/Th17/Treg/Tfh、CD8 は Tem/Tcm/Trm/exhausted/MAIT を内包。MS で異常を示すのは Th17 や CD8 effector のような数 % のサブセットのみ。

② Pseudobulk dilution: donor 内の全細胞平均で サブセット信号が 90% の "通常" T 細胞に薄められる。

③ edgeR の harsh BH-FDR: 細胞数が多いと dispersion 推定が tight になり、わずかな差は検出されるが多重補正で脱落。

④ Biology filter: 残った "わずかな差" は ribosomal/HK 系遺伝子が多く、これを除外すると候補ほぼゼロ。
▶ 重要な解釈 — これは CD4/CD8 が MS と無関係であることを意味しない
CD4 Th17 の IL-17 経路、CD8 cytotoxic の demyelination、Treg 機能不全は MS 病態の中心軸です (Ramesh 2020, Schafflick 2020, IL7R/IL2RA は MS GWAS top hit)。 本研究の現状 panel は 「壁」の biology (B/Mono/NK の axes) は捕捉できたが、 「中心メカニズム」(CD4 Th17, CD8 cytotoxic) は pseudobulk dilution で見えていないのが正直な状態。
▶ Phase 2 で MIL が必須である理由: Multi-Instance Learning (MIL) は cell-level 解像度で attention 機構により 「どの細胞 subset が disease を駆動しているか」を pseudobulk なしで識別します。 CD4 内の Th17、CD8 内の Trm/exhausted を attention weight として浮かび上がらせる。 本研究の最大の制約 (T 細胞 dilution) を直接解消するのが Phase 2 MIL の存在理由です。
